## Project setup

1. Clone repository

```bash
    git clone git@github.com:Raphico/personal-finance-app.git
    cd personal-finance-app
```

2. Install dependencies

```bash
    pnpm install
```

3. setup env
4. start development server

```bash
  pnpm run dev
```
